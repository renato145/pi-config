Write a concise conventional commit message for the latest changes you made

Rules:

- Format: type: subject
- Type: feat, fix, docs, style, refactor, perf, test, or chore
- Subject: imperative mood, lowercase, no period, max 50 chars
- Body: only if the "why" isn't obvious, wrap at 72 chars
